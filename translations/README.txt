# Traductor (Translator)

## Description
Traductor is a simple translation tool that allows users to translate text between different languages. It utilizes the Google Translate API to provide accurate translations.

## Features
- Translate text from one language to another
- Support for multiple languages
- User-friendly interface
- Fast and accurate translations

## Installation
1. Clone this repository:
   ```
   git clone https://github.com/yourusername/traductor.git
   ```
2. Navigate to the project directory:
   ```
   cd traductor
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage
1. Run the main script:
   ```
   python traductor.py
   ```
2. Follow the on-screen prompts to enter the text you want to translate, the source language, and the target language.

## Dependencies
- googletrans==3.1.0a0
